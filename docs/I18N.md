# Internationalization

ArenaForge now includes a locale catalog foundation in
`arena_forge/locales/`.

## Principles

- Source files, metadata, and provider parsing are UTF-8 first.
- Paths and persisted test sessions use normal filesystem files instead of
  platform-specific alternate data streams.
- Translatable strings are keyed, not embedded.
- Locale lookup falls back to English when a key is missing.

## Current Locales

- `en`
- `zh-Hans`
- `ja`
- `ko`
- `ru`

## Next Steps

- Extend static package resources such as menus and command captions with locale-aware variants.
- Add locale selection to the package settings UI.
- Expand provider and diagnostics coverage further as new command surfaces are added.
